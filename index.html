gigapipe Helm Charts

This GitHub Pages site hosts a Helm repository for gigapipe.

Usage

Add this repo to Helm:

helm repo add gigapipe https://metrico.github.io/gigapipe-helm/

Update your local cache:

helm repo update

Install the chart:

helm install gigapipe gigapipe/gigapipe

Repository Contents

  • index.yaml - Helm repository index
  • charts/ - Packaged charts
